How are we different?

We created this software with the intention of making the day-to-day work flow of creative individuals easier. The QWS Software Foundation wishes to maintain the QWS open source and has no plans to monetise it. QWS was designed with the user's privacy in mind, therefore it does not disclose any information online with any third party and can also run offline.

You don't have to be a developer to understand how the QWS works; we've simplified it for you!
